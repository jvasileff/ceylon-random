LCGRandomshared final LCGRandom

A Linear Congruential Generator (LCG) pseudorandom number generator, defined by the recurrence relation:

Xₙ₊₁ ≡ (a Xₙ + c) (mod m)

Using the parameters:

a = 25214903917
c = 11
m = 2^48

Note: for the JavaScript runtime, there is a loss of precision for Integers greater than 253 which may result in decreased quality of random numbers produced by this class.

See http://en.wikipedia.org/wiki/Linear_congruential_generator
